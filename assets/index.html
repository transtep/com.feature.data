feature.transtep.data 教學範例

data.channel 料道物件

名稱 方法 描述 範例
quantity get 傳入料道編號取得該料道目前存量
  list 取得所有料道存量
product get 取得料道對應商品
  list 取得所有料道對應商品
price get 傳入料道編號取得價格
  list 取得料道對應價格
status get 傳入料道編號取得狀態
  list 取得料道狀態

data.payment 支付物件

方法 描述 範例
get 取得對應的支付資訊
list 取得所有支付資訊

data.product 商品物件

方法 描述 範例
get 以商品id取得商品資料
list 取得販賣機所有商品資料

data.property 機台設置物件

方法 描述 範例
get 取得client販賣機參數值
list 取得client販賣機所有參數

使用方式: 引入 feature.transtep.data 函式庫

於 plugin/config.xml 加入 <feature name="data" value="feature.transtep.data"/>
